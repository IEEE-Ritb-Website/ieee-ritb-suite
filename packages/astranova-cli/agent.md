# 📦 astranova-cli (Shared Package)

## 📝 Description
Developer scaffolding tool and microservice generator.

---

## 🎯 Scope
Exposes template builders to instantly scaffold new microservices in the monorepo (Express backends with `pnpm rs create-be` and Vite+Tailwind frontends with `pnpm rs create-fe`), maintaining consistent settings and formatting structures.

---

## 🛠️ Usage & Integration
```typescript
pnpm rs create-be my-service
pnpm rs create-fe my-app
```
