# 📦 admin-service (Backend Microservice)

## 📝 Description
Authentication and user management backend gateway.

---

## 🎯 Scope
Implements `better-auth` controllers, user session storage, database collections mapping, and administrative roles verification. Connects to MongoDB and Cloudinary.

---

## 🛠️ Usage & Integration
```typescript
Runs on Express.js v5. Managed via standard MVC structure inside `src/`.
```
