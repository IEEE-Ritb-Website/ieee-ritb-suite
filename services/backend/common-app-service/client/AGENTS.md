# 📦 @astranova/common-app-client (Client Library)

## 📝 Description
Strongly-typed Axios client SDK for common-app-service.

---

## 🎯 Scope
Wraps developer tools and URL shortener microservice HTTP routes in clean TypeScript interfaces, facilitating easy frontend integration.

---

## 🛠️ Usage & Integration
```typescript
import { CommonAppClient } from "@astranova/common-app-client";
```
