# 📦 form-service (Backend Microservice)

## 📝 Description
Form submission and event registration processing API.

---

## 🎯 Scope
Captures and processes public event registrations, runs Zod request validations, and handles MongoDB collections storage.

---

## 🛠️ Usage & Integration
```typescript
Runs on Express.js v5. Utilizes schema validation rules.
```
