# 📦 common-app-fe (Frontend Application)

## 📝 Description
Developer utilities dashboard client.

---

## 🎯 Scope
Vite-based React SPA styled with Tailwind CSS, offering interactive tools (UUID/hash generation, JSON parse tools) and calling common-app-service backend APIs.

---

## 🛠️ Usage & Integration
```typescript
pnpm dev (boots dev server with Hot Module Replacement)
```
