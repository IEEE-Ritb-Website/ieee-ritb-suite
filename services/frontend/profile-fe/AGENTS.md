# 📦 profile-fe (Frontend Application)

## 📝 Description
Next.js 16 user dashboard and conclave profiles manager.

---

## 🎯 Scope
Server-Side Rendered dashboard, managing auth cookies, student conclave registrations, profile verification forms, and email triggers via Nodemailer.

---

## 🛠️ Usage & Integration
```typescript
pnpm dev (Next.js server-side development mode)
```
